### 🔬 Phishing Email Investigation & Payload Decoding

Used CyberChef to decode Base64-obfuscated payloads, confirmed hostile malicious file signatures using HxD Hex Editor, and extracted hidden attacker infrastructure using ExifTool to output a clean Indicators of Compromise (IOC) report.
